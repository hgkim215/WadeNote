# WadeNote 개인정보 처리방침

최종 업데이트: 2026-06-23

> 이 문서는 초안입니다. App Store 제출 전 **공개 URL로 호스팅**해야 하며
> (예: GitHub Pages, 본인 웹사이트), App Store Connect의 "개인정보 처리방침 URL"에
> 입력해야 합니다. 회사명·연락처·관할 법 등은 본인 상황에 맞게 채워 주세요.

## 1. 개요

WadeNote("앱")는 사용자가 비밀번호·계정·카드/은행 정보·신분증 정보·보안 메모 등
민감한 개인정보를 **사용자 본인의 기기와 본인의 iCloud에만** 안전하게 보관하도록 돕는
앱입니다. 개발자는 사용자의 데이터를 수집·열람·전송·판매하지 않습니다.

## 2. 수집하는 정보

- **개발자가 수집하는 정보: 없음.** 앱은 분석 도구, 광고 SDK, 추적 기술을 포함하지
  않으며, 사용자 데이터를 개발자 서버나 제3자에게 전송하지 않습니다.
- **사용자가 앱에 직접 입력·저장하는 정보**: 사용자가 저장하기로 선택한 항목
  (서비스 계정, 비밀번호, 카드/계좌 번호, 신분증 번호, 메모, 첨부 사진 등).
  이 정보는 전적으로 사용자의 통제 하에 있습니다.

## 3. 정보의 저장 위치와 보안

- **로컬 저장**: 데이터는 기기의 앱 저장소에 저장되며 iOS의 기기 암호화로 보호됩니다.
- **iCloud 동기화(선택)**: 사용자가 iCloud에 로그인한 경우, 데이터는 **사용자 본인의
  비공개 iCloud(CloudKit private database)**에 저장·동기화되어 다른 기기에서 복원됩니다.
  이 데이터는 Apple이 전송·저장 시 암호화하며, 사용자가 **고급 데이터 보호(ADP)**를
  활성화하면 종단간 암호화됩니다. 개발자는 이 데이터에 접근할 수 없습니다.
- **첨부 사진**: 기기에 저장되기 전에 앱이 추가로 AES-GCM으로 암호화하며,
  암호화 키는 기기의 Keychain(보안 영역)에 보관됩니다.
- **앱 잠금**: 앱 진입 시 Face ID/Touch ID 또는 기기 암호로 잠금을 해제해야 합니다.

## 4. 권한 사용

- **Face ID / Touch ID**: 앱 잠금 해제 용도로만 사용합니다. 생체정보는 Apple이 관리하며
  앱은 그 값에 접근하지 않습니다.
- **사진**: 사용자가 항목에 사진을 첨부할 때 iOS 사진 선택기(PhotosPicker)를 통해
  사용자가 직접 고른 사진만 가져옵니다. 앱은 전체 사진 라이브러리에 접근하지 않습니다.

## 5. 정보의 제3자 제공

개발자는 어떤 사용자 정보도 제3자에게 제공·판매·공유하지 않습니다. iCloud 동기화는
Apple의 iCloud 서비스를 통하며, 이는 [Apple 개인정보 보호정책](https://www.apple.com/legal/privacy/)을
따릅니다.

## 6. 데이터 보관 및 삭제

- 사용자는 앱 내에서 언제든 항목을 삭제할 수 있습니다.
- 앱을 삭제하면 기기의 로컬 데이터가 제거됩니다.
- iCloud에 저장된 데이터는 사용자의 iCloud 계정 설정에서 관리·삭제할 수 있습니다.

## 7. 아동의 개인정보

이 앱은 만 13세(또는 해당 관할의 기준 연령) 미만 아동을 대상으로 하지 않으며,
아동의 개인정보를 알면서 수집하지 않습니다.
