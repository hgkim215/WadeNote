# WadeNote — 디자인 브리프 (클로드 디자인용)

> 이 문서는 UI 디자인 생성 도구에 첨부하기 위한 디자인 브리프다.
> 설계 문서(`2026-06-22-wadenote-design.md`)의 비주얼·UX 측면을 디자인 생성에 바로 쓸 수 있게 구체화한다.

## 1. 한 줄 소개

민감한 개인정보(계정/비밀번호, 카드/은행, 신분증, 보안 메모)를 **유형별 카드로 구조화**해 한눈에 읽히게 보관하고, Face ID로 잠그며 iCloud로 모든 기기에 복원되는 **iOS 네이티브 앱**.

## 2. 디자인 원칙

1. **가독성 최우선** — 빽빽한 텍스트 메모가 아니라, 라벨·값이 또렷하게 구분되는 구조화된 카드. 훑어보기만 해도 "이게 무슨 정보인지" 즉시 파악.
2. **신뢰감 있는 보안 톤** — 차분하고 단정한 분위기. 금고/방패 같은 과장된 메타포는 절제하고, 차분한 색·여백·명확한 위계로 "안전하게 관리되고 있다"는 느낌.
3. **빠른 동작** — 가장 잦은 행동(찾기 → 복사)을 최소 탭으로. 빠른 복사는 항상 손가락이 닿는 곳에.
4. **유형별 시각 구분** — 4개 유형을 색/아이콘으로 일관되게 구분해 목록에서 스캔이 쉬움.

## 3. 비주얼 스타일

- **플랫폼 관용**: iOS 네이티브 감성 (SF Pro 폰트, 시스템 여백, 큰 타이틀 네비게이션, 둥근 카드, 햅틱 톤). 라이트/다크 모드 모두 지원.
- **레이아웃**: 카드 기반. 넉넉한 여백, 명확한 섹션 구분.
- **타이포 위계**: 항목 제목(강조) > 필드 라벨(작고 흐림) > 필드 값(읽기 쉬움). 비밀값은 모노스페이스 ••••.
- **컬러**: 차분한 뉴트럴 배경 + 유형별 액센트 컬러 1개씩.
  - 로그인 — 블루
  - 카드/은행 — 그린
  - 신분증 — 퍼플
  - 보안 메모 — 그레이/뉴트럴
- **아이콘**: SF Symbols 계열. 유형별 대표 심볼 (로그인=자물쇠/사람, 카드=신용카드, 신분증=신분증, 보안메모=문서).
- **상태 표현**: 비밀값 가림/보임, 복사됨 토스트, 즐겨찾기 별, 동기화 상태(작게).

## 4. 화면별 상세

### 4.1 잠금 화면
- 앱 진입/재진입 시 첫 화면.
- 중앙에 앱 로고/이름, Face ID 안내 아이콘.
- "Face ID로 잠금 해제" 상태 표시. 실패 시 "패스코드 입력" 폴백 버튼.
- 배경은 차분하고, 잠금 중엔 어떤 데이터도 노출 안 됨.

### 4.2 목록(홈)
- **큰 타이틀** "WadeNote" + 우상단 추가(+) 버튼.
- 상단 **검색바**.
- **즐겨찾기 섹션**(별 표시 항목) — 상단 고정, 가로 스크롤 또는 강조 카드.
- 그 아래 **전체 목록** — 유형별로 그룹핑하거나 최근순 단일 리스트(둘 다 디자인 시안 필요).
- **각 행(아이템 카드)**:
  - 좌측: 유형 아이콘(유형 색 적용)
  - 중앙: 제목 + 대표 부가정보(예: 로그인이면 아이디 일부)
  - 우측: 즐겨찾기 별, "대표 필드 빠른 복사" 버튼
- 비어 있을 때: 친절한 빈 상태 일러스트 + "첫 정보 추가하기" CTA.

### 4.3 상세 화면
- 상단: 제목 + 유형 뱃지 + 즐겨찾기 토글 + 편집 버튼.
- **구조화된 필드 카드 리스트**: 각 행 = 라벨 + 값 + 우측 복사 버튼.
  - 비밀값 필드: 기본 ••••, 눈 아이콘 탭하면 보임/가림.
  - URL 필드: 탭하면 열기.
  - 날짜 필드: 읽기 쉬운 포맷.
- **첨부 사진**: 하단에 썸네일 그리드, 탭하면 전체 화면 뷰어.
- 메타: 생성/수정일 작게.

### 4.4 편집/추가 화면
- 신규: 먼저 **유형 선택**(4개 카드 중 선택) → 템플릿 필드 자동 생성.
- 각 필드 입력 폼: 라벨 + 입력 칸 + 타입에 맞는 키보드/마스킹.
- **사용자 정의 필드 추가** 버튼.
- **사진 첨부** 영역(추가/삭제).
- 저장/취소.

## 5. 핵심 인터랙션 (마이크로 UX)

- **빠른 복사**: 탭 → 가벼운 햅틱 → "복사됨" 토스트 → N초 후 클립보드 자동 삭제(사용자에게는 조용히).
- **비밀값 토글**: 눈 아이콘으로 보임/가림. 일정 시간 후 자동 재가림 고려.
- **즐겨찾기**: 별 탭 → 즉시 홈 상단 반영.
- **검색**: 입력 즉시 필터링. 제목·유형·필드 라벨 대상(필드 값은 보안상 제외).

- 보안 메모 — "와이파이 비밀번호 모음" / 본문(자유 텍스트)

## 7. 필요한 디자인 산출물 (시안)

1. 잠금 화면 (라이트/다크)
2. 목록(홈) — 즐겨찾기 + 유형별 그룹 / 단일 리스트 두 가지 시안
3. 빈 상태(empty state)
4. 상세 화면 (로그인 유형 예시, 비밀값 가림/보임 상태 둘 다)
5. 편집/추가 — 유형 선택 + 입력 폼
6. 빠른 복사 토스트, 비밀값 토글 등 마이크로 상태
7. 유형별 아이콘/컬러 시스템 + 컴포넌트 (아이템 카드, 필드 행, 뱃지, 버튼)
