# WadeNote — 비주얼 디자인 시스템 (구현용)

- 작성일: 2026-06-22
- 출처: 클로드 디자인 프로젝트 "WadeNote iOS 앱 UI 시안" (projectId `679173b2-7839-4875-bf02-5f142b1ddaaa`)
- 채택 시안: `WadeNote v2.dc.html` (라이트 퍼스트 리디자인) + `WadeNote App Icon.dc.html`
- 관련 문서: `2026-06-22-wadenote-design.md`(설계), `2026-06-22-wadenote-design-brief.md`(브리프)

## 1. 방향성

애플 기본앱의 평면적 그룹 리스트에서 벗어나 — 컬러 그라데이션 타일 + 부드러운 컬러 섀도, 헤어라인 보더 + 레이어드 섀도로 깊이감, 상단에 은은한 브랜드 배경광(radial glow). **라이트 퍼스트**, 따뜻한 뉴트럴 배경. 유형 4색 유지하되 더 선명한 톤. 비밀값은 모노스페이스.

## 3. 형태·간격

- 폰트: SF Pro (`-apple-system, system-ui`). 비밀값은 `ui-monospace, monospace` + letter-spacing 1px.
- 라운드: 카드 16px, 아이콘 타일 11~12px, 작은 환형 버튼 50%, 큰 로고 타일 19~24px.
- 카드 섀도(라이트): `0 1px 2px rgba(20,20,40,.04), 0 14px 30px -16px rgba(20,20,40,.18), inset 0 0 0 .5px rgba(20,20,40,.04)`.
- 헤어라인 보더: `inset 0 0 0 .5px ...` (1px 미만 느낌).
- 화면 패딩: 좌우 22px.

## 4. 컴포넌트 사양

- **아이템 카드(행)**: [그라데이션 타일 36~38px] + [제목 15px/600 + 서브 12.5px] + (즐겨찾기 별) + [환형 복사 버튼 30px, 배경 `rgba(61,116,255,.1)`].
- **그라데이션 아이콘 타일**: 유형 컬러 그라데이션 + 컬러 섀도. 흰색 stroke 아이콘(stroke-width 2).
- **필드 행**: [라벨 11px `#a0a0ac` + 값 15~16px] + (비밀값이면 모노스페이스 ••••) + 눈 토글 + 환형 복사 버튼. 행 구분 `.5px solid rgba(20,20,40,.07)`.
- **즐겨찾기 카드**: 유형색 옅은 그라데이션 배경 `linear-gradient(160deg, <accent .13>, <accent .03>)` + `inset 0 0 0 .5px <accent .18>`. 타일 + 제목 + 서브. 가로 스크롤.
- **유형 뱃지**: pill, `background <accent .12>`, `color <accent>`, `inset 0 0 0 .5px <accent .25>`.
- **기본 버튼(저장)**: 유형/액션 그라데이션 + `0 6px 16px -4px <accent .6>` 섀도, 흰 텍스트 14px/600, 라운드 12px.
- **동기화 뱃지**: pill, 작은 그린 iCloud 아이콘 + "iCloud 동기화됨".
- **복사 토스트**: 하단 중앙, 페이드+업 애니메이션(`wnToast`), 다크 pill.

## 5. 화면별 레이아웃

상단 공통: iOS 상태바(9:41 + 신호/와이파이/배터리), Dynamic Island(120×33 노치), 하단 홈 인디케이터. 기기 프레임 393×852, 라운드 52px.

### 5.1 잠금 화면
- 중앙: 큰 로고 타일 92px(다크 그라데이션 `#3c3c46→#16161a`에 흰 노트 글리프) + "WadeNote" 27px/700 + "잠금됨 · 데이터 비공개".
- 하단: 환형 Face ID 아이콘(블루) + "Face ID로 잠금 해제" + "기기를 바라보면 자동으로 열려요" + "패스코드 입력" 보조 버튼.

### 5.2 홈 — 즐겨찾기 + 유형별 그룹
- 큰 타이틀 "WadeNote" 33px/700 + 그 아래 "iCloud 동기화됨" 뱃지.
- 우상단 + 버튼(그라데이션 환형).
- 검색바(흰 카드, 돋보기 + "검색").
- **즐겨찾기** 섹션: 가로 카드들(유형색 옅은 카드).
- **유형별 그룹**: 섹션 라벨(대문자 트래킹) + 흰 카드 안에 아이템 행들. 각 행 우측 즐겨찾기 별 + 복사 버튼.

### 5.3 상세 — 동작(눈/복사)
- 네비: "‹ 목록" / "편집".
- 헤더: 큰 그라데이션 타일 72px + 제목 25px/700 + 유형 뱃지 + 즐겨찾기 토글(별).
- 필드 카드: 아이디(값+복사), 비밀번호(모노 •••• + 눈 토글 + 복사). 눈 토글 시 값 표시/마스킹, 복사 시 토스트.
- (하단) 첨부 사진 썸네일 그리드(설계 문서 기준).

### 5.4 추가 — 입력 폼
- 유형 선택 → 템플릿 필드 자동 생성 → 값 입력. 저장 버튼(그라데이션).

### 5.5 다크 모드
모든 화면 다크 버전 제공. 배경 `#0c0c0e`, 카드 `#161619`, 텍스트/보조색 위 토큰표 다크 열 적용. 그라데이션 타일·섀도는 동일 톤 유지.

## 6. 앱 아이콘 (채택)

- 컨셉: **흰 노트 페이지(메모)** + 가려진 **•••• 점(프라이버시)** + 블루 **잠금 뱃지(보안)**를 한 마크에. 글리프 동일, 배경만 바꾼 라이트/다크 적응형.
- 글리프 SVG (viewBox 0 0 24 24):
  - 노트 본체: `rect x=4.6 y=3.3 w=14.8 h=17.4 rx=3.3 fill=#fff`
  - 제목 줄: `rect x=7.2 y=6.9 w=7 h=1.7 fill=#1c1c22`
  - 블루 줄: `rect x=7.2 y=10.6 w=9 h=1.5 fill=#0A84FF`
  - •••• 점 3개: `circle cy=14.7 fill=#c4c4ce` (cx 7.9 / 10.1 / 12.3)
  - 잠금 뱃지: `rect x=12.6 y=12.8 w=8.4 h=8.4 rx=2.7 fill=#0A84FF` + 흰 자물쇠 몸통/고리
- 배경:
  - 라이트: `linear-gradient(160deg,#eceef3,#d7dbe5)`
  - 다크: `linear-gradient(160deg,#2c2c34,#0e0e11)`
- 앱 내 로고 타일은 다크 그라데이션(`#3c3c46→#16161a`) + 블루 줄 `#3D74FF` 변형 사용.
- 산출: iOS 앱 아이콘은 단일 적응형 마크 기준으로 1024 마스터 + 필요 사이즈. 글리프는 위 SVG를 자산으로 사용.

## 7. 디자인 ↔ 설계 매핑 메모

- 시안의 `sc-for`/`sc-if`/`{{ }}`는 클로드 디자인 템플릿 문법 — 데이터 바인딩 위치를 나타냄. SwiftUI 구현 시 `ForEach`/조건 분기/모델 프로퍼티로 치환.
- 유형 4색·아이콘은 설계 문서의 유형 4종(로그인/카드·은행/신분증/보안 메모)과 1:1 대응 → `Templates` 모듈에 색·심볼 토큰으로 정의.
- 비밀값 모노스페이스 + 눈 토글 + 복사 토스트는 설계의 "비밀값 가림/빠른 복사(N초 자동삭제)"와 대응.
