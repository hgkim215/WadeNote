WadeNote란?

내 모든 개인정보를 한곳에. 유형별 템플릿으로 정리하고, Face ID로 잠그며, iCloud로 모든 기기에 복원하는 iOS 개인정보 금고 앱입니다. 정보는 본인 기기와 본인 iCloud에만 저장되며, 추적·광고·외부 서버 전송이 없습니다.

  • 유형별 템플릿 — 로그인 · 카드/은행 · 신분증 · 보안 메모
  • Face ID 잠금 + 앱 전환 화면 가림
  • iCloud(CloudKit)로 새 기기에서 그대로 복원
  • 빠른 복사 후 30초 자동 삭제, 이 기기에만 보관
  • 검색 · 즐겨찾기 · 암호화 사진 첨부

자주 묻는 질문

기기를 바꾸면 데이터가 사라지나요?
아니요. 같은 Apple ID로 iCloud에 로그인한 뒤 앱을 설치하면 저장한 항목이 자동으로 복원됩니다.
제 정보를 개발자가 볼 수 있나요?
볼 수 없습니다. 데이터는 본인 기기와 본인의 비공개 iCloud에만 저장되며 개발자 서버로 전송되지 않습니다.
Face ID가 안 될 때는요?
Face ID 실패 시 기기 암호(패스코드)로 잠금을 해제할 수 있습니다.
더 강력한 암호화를 원해요.
iCloud 설정에서 고급 데이터 보호(ADP)를 켜면 동기화 데이터가 종단간 암호화됩니다.
